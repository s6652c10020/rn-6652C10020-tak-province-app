* 📱 รายละเอียดโปรเจกต์โปรเจกต์นี้เป็นส่วนหนึ่งของวิชา Mobile Application Development 
โดยพัฒนาด้วย React Native (Expo) และใช้ Supabase เป็นระบบหลังบ้าน (Backend) ในการจัดเก็บข้อมูลสถานที่ท่องเที่ยวในจังหวัดตากครับ🛠️

_____________________________________________________________________________________________________

* ฟีเจอร์หลักของแอปหน้า Home :
แสดงข้อมูลเบื้องต้นของจังหวัดตาก พร้อมรูปภาพหน้าปกที่ดึงจากไฟล์ในเครื่อง (assets/images) 
มีการแบ่งหมวดหมู่การค้นหาชัดเจน 5 หมวดหน้า List: ดึงข้อมูล Real-time จาก Supabase ตามหมวดหมู่ที่เลือก (ที่เที่ยว, ร้านอาหาร, คาเฟ่, วัด, และงานประเพณี)

_____________________________________________________________________________________________________

* หน้า Detail : แสดงรายละเอียดเชิงลึกของแต่ละสถานที่ พร้อมรูปภาพที่ดึงจาก URL และมีระบบแผนที่ (Google Maps) 
ที่สามารถกดนำทางไปยังพิกัดจริงได้Cross-Platform: แอปสามารถรันได้ทั้งบน iOS, Android และ Web Browser (PC) โดยมีการดักจับ Error กรณีรันบนเว็บเพื่อให้แอปไม่ค้างครับ🗄️ 

_____________________________________________________________________________________________________

  โดยมีการเก็บข้อมูล Supabase ดังนี้
